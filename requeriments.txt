Bibliotecas necessárias para o código

Para instalar basta abrir o prompt de comando/powershell (ou digitar no terminal do vscode) e digitar:

python -m pip install bs4
python -m pip install pandas
python -m pip install requests
python -m pip install openpyxl
python -m pip install plotly
python -m pip install kaleido
python -m pip install flet


Espere a instalação de cada um terminar cara executar a proxima instalação
